## 使用

### 方法一：Overleaf（推荐）

1. 在 [Overleaf](https://www.overleaf.com) 注册账号并新建项目
2. 将本文件夹所有文件上传
3. 直接编译即可使用

### 方法二：本地 TeX 环境

需要安装以下环境：
- TeX Live（Windows/Linux/macOS）或 MiKTeX（Windows）
- 中文字体（思源黑体 / NOTO Sans CJK）

编译命令：
```bash
xelatex slide.tex
```
